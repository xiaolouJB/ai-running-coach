## 二、第一步：连接你的运动手表

> 只需做一次，之后每次打开 Claude 自动生效。

### 2.1 连接高驰 MCP（约 3 分钟）

**第 1 步**：打开 Claude Code 桌面版，在对话框输入：

```
/mcp
```

**第 2 步**：如果列表里没有 `coros`，继续下一步；如果已有，跳到第 4 步。

**第 3 步**：告诉 Claude：

```
帮我安装高驰 COROS MCP，地址是 https://mcpcn.coros.com/mcp
```

Claude 会自动完成配置，并提示你**重启**一次。

**第 4 步**：重启后再次输入 `/mcp`，看到 `coros` 出现即成功。

**第 5 步**：首次使用会弹出一个**浏览器登录页面**，用你的高驰账号（邮箱+密码）登录授权即可。

> ✅ 连接成功后，Claude 就能直接读取你手表里的所有运动数据，无需再手动上传任何文件。

## 三、第一次使用：生成你的专属课表

### 3.1 激活 AI 教练系统

将以下内容**完整复制**发送给 Claude（修改 `[ ]` 内的内容）：

```
请帮我初始化 AI 跑步教练系统。

我的信息：
- 目标比赛：[半程马拉松 / 全程马拉松 / 10公里 / 5公里]
- 目标完赛时间：[例如 1小时30分]
- 每周可以训练的天数：[例如 4天]
- 目标比赛日期：[例如 2026年11月15日]
- 每周跑量上限（不填则不限）：[例如 60公里，或者不限]
- 我的历史最佳成绩（不填则从手表自动读取）：[例如 半马1:38，或者不填]

请读取我高驰手表最近6周的数据，生成我的专属训练计划。
```

### 3.2 AI 会自动做什么

发送后，Claude 会：

1. 🔍 读取你手表里近 6 周的跑步数据（自动，无需操作）
2. 📊 分析你的 VO2max、配速区间、恢复能力
3. 📅 生成从今天到比赛日的完整训练计划
4. 💾 保存到 `schedules/你的昵称_schedule.md` 文件中

整个过程约 **1-2 分钟**，等待即可。

## 四、日常使用：每周一句话更新

这是这个教练系统**最核心的功能**，也是最简单的操作。

### 4.1 每周训练结束后（推荐每周日晚上）

直接对 Claude 说：

```
更新本周训练数据
```

Claude 会自动：
- 从手表读取本周所有跑步记录
- 给每次训练打分、写评语
- 分析本周整体训练质量
- 根据你的实际表现，判断是否需要调整下周训练计划

**就这一句话，其他什么都不用做。**

> **关于训练日期**：不用担心实际训练日期和课表不一致。AI 是按照「本周训练类型是否完成」来判断的，而不是严格按日期对位。比如课表说周四跑 T 跑，但你周六才跑，AI 依然会正确识别并评分。

### 4.2 随时复盘（可选）

除了每周例行更新，你还可以随时让 AI 复盘训练：

| 你想了解的 | 怎么说 | AI 分析范围 |
|-----------|--------|-----------|
| 刚跑完，想看分析 | "帮我分析一下今天那次跑步" | 最近1条活动 |
| 本周整体怎么样 | "复盘一下这周的训练" | 本次更新数据 + 本自然周 |
| 最近状态趋势 | "我最近两周状态怎么样" | 最近14天（可指定天数）|

### 4.3 关于课表调整频率

AI **不会**因为你某一天状态不好就立刻改课表——一次表现差可能只是睡眠、天气、心情的原因，并不代表训练计划有问题。

AI 按三档判断是否调整：

| 调整级别 | 触发条件 | 说明 |
|---------|---------|------|
| 紧急保护（每日）| HRV 大幅下降 或 静息心率异常升高 | 立即插入恢复日，防止过训 |
| 常规调整（每周）| 本周多次训练质量偏低 | 下周配速微调，幅度 ±5~10s/km |
| 周期重评（每4周）| 达到4周周期边界 | 重新评估 VDOT，必要时更新全部配速区间 |

每次调整都会在课表的「变更记录」表格中写明原因和理论依据，你可以随时查看。

### 4.4 临时想了解某些信息

你可以随时问 Claude：

| 你想知道的 | 怎么说 |
|-----------|--------|
| 今天适合训练吗 | "今天适合跑步吗？" |
| 昨晚睡眠怎么样 | "我昨晚睡得好不好？" |
| 距离目标还差多少 | "我现在的状态能跑多少时间？" |
| 明天跑什么 | "明天的训练是什么？" |
| 查看课表变更 | "这周课表有没有调整，为什么？" |

## 六、如何修改个人配置

### 6.1 修改训练目标或日期

直接告诉 Claude：

```
我把目标时间改成1小时28分，比赛日期改成2026年12月1日
```

Claude 会更新你的课表文件，并重新规划后续计划。

### 6.2 修改数据拉取范围

默认拉取**最近6周**数据建立基线，**最近7天**数据做每周更新。  
如果想改，告诉 Claude：

```
把历史数据拉取范围改成最近3个月
```

或者直接打开 `schedules/你的昵称_schedule.md` 文件顶部，修改参数：

```yaml
data_scope:
  baseline_weeks: 12     # 改成你想要的周数
  weekly_update_days: 7  # 每周更新拉取天数，一般不用改
  auto_overwrite: true   # 改成 false 则每次更新前会询问你
```

### 6.3 手动覆盖你的历史 PB

如果手表里没有你的比赛记录，可以手动告诉 Claude：

```
我的历史半马成绩是1小时34分，用这个作为我的基线
```

### 6.4 修改每周跑量限制

```
我每周最多只能跑50公里，帮我调整课表
```

### 6.5 临时跳过某周训练

如果这周有事没法跑，告诉 Claude：

```
这周我出差了，基本没跑，下周课表不要在这周基础上递增
```

## 七、常见问题 FAQ

**Q：高驰 MCP 连接后，每次打开 Claude 都需要重新登录吗？**  
A：不需要。首次授权后，token 会自动保存。但如果长时间不用（超过几个月）可能需要重新授权一次。

**Q：我的数据安全吗？Claude 会存储我的运动数据吗？**  
A：Claude 只在当次对话中读取和处理你的数据，不会永久存储。高驰 MCP 通过官方 OAuth 授权，符合标准安全规范。

**Q：课表文件在哪里？如何查看？**  
A：在 `ai-running-coach/schedules/你的昵称_schedule.md`，用任何文本编辑器（记事本、VS Code 等）都能打开查看。

**Q：我不在课表安排的日期训练了怎么办？**  
A：没关系，直接跑完后告诉 Claude "更新本周数据"，AI 会根据你实际跑的内容来评估，不会死板地按日期比对。

**Q：课表里的配速看不懂怎么办？**  
A：问 Claude，例如："课表里的 E 跑 5:30-5:50 是什么意思，我应该跑多快？"

**Q：我实际训练的日期和课表不一样，AI 会扣分吗？**  
A：不会。AI 按照"本周这种类型的训练是否完成"来评估，不管你在周几完成。比如课表安排周四 T 跑，你周六才跑，AI 依然会正确匹配并评分。只要本周计划的训练类型都完成了，就视为完成。

**Q：昨天状态很差，AI 会因为这一次就改我的课表吗？**  
A：不会。AI 只在以下情况才调整课表：①生理异常信号（HRV 大幅下降或静息心率异常，立即保护）；②本周多次训练质量持续偏低（每周评估）；③每4周的周期重评。单次状态差不会触发调整，AI 会结合你的睡眠、HRV 等数据综合判断。

**Q：课表被调整了，怎么知道为什么改的？**  
A：每次调整都会在课表文件顶部的「变更记录」表格中写明日期、触发原因、具体调整内容和对应的训练理论依据。你可以随时打开课表文件查看，或者问 Claude："这周课表为什么调整了？"

**Q：复盘和每周更新有什么区别？**  
A：每周更新（"更新本周训练数据"）是标准流程，会写入课表、更新状态、判断是否调整下周计划。复盘（"复盘一下""分析今天"）是按需触发的深度分析，只输出报告，不强制更新课表，但如果触发了调整条件，同样会写入变更记录。

**Q：AI 复盘结束时问我感受是什么意思？**  
A：高驰 MCP 目前只能获取客观数据（配速、心率等），无法读取你在手表上记录的主观感受。所以 AI 会在复盘结束时询问你"这次感觉怎么样，1-10分"，这个信息会被保存在课表的「感受」列中，帮助 AI 更准确地评估训练质量。如果你不想被问，可以让 Claude 关掉这个功能。

**Q：我用佳明手表，能用吗？**  
A：目前佳明适配正在开发中，暂不支持自动拉取数据。但你可以手动告诉 Claude 你的训练数据，AI 仍然能帮你分析和调整课表。

**Q：网页版 Claude 能用吗，还是必须装 Claude Code？**  
A：网页版也可以用，但 Claude Code 桌面版支持直接读写本地的课表文件（`schedule.md`），体验更好。

**Q：可以把这个 Skill 分享给朋友用吗？**  
A：可以！把整个 `ai-running-coach` 文件夹分享给他们，让他们按照本手册第二步重新连接自己的高驰账号即可。

## 八、注意事项

### ⚠️ 使用前必读

1. **数据来源**：AI 教练的所有分析基于你高驰手表记录的数据。数据越丰富（跑步历史越长），分析越准确。

2. **不能替代医嘱**：如果你有伤病、心脏病或其他健康问题，请先咨询医生，不要完全依赖 AI 建议进行高强度训练。

3. **配速仅供参考**：AI 推算的配速区间基于数据模型，实际跑步时请结合自身感受，身体不适随时停止。

4. **课表会动态变化**：每次更新后，下周课表可能会有调整，建议每周日查看一次最新课表，不要提前太久安排。

5. **保持数据同步**：每次跑完后记得让手表数据同步到高驰 App，这样 Claude 才能读取到最新数据。

### 💡 使用建议

- **最佳更新时机**：每周日晚上，训练结束并同步数据后
- **最佳提问方式**：直接用自然语言描述，不需要任何格式
- **遇到不对的建议**：直接告诉 Claude "这个配速对我来说太快了"，AI 会重新调整
- **长期使用**：坚持每周更新，AI 对你的了解会越来越深，建议会越来越精准
