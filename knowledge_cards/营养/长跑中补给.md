# 长跑中补给（15km+ / 比赛日）

> 来源：Suzanne Girard Eberle《Endurance Sports Nutrition》（ACSM 认证运动营养师）

## 补给触发时机

**AI 触发规则**：
- 首次完成 >15km 长跑 → 提示能量胶基础知识
- 首次完成 >20km 长跑且气温 >20°C → 引导测量出汗率

---

## 碳水化合物补充

**ACSM 标准**：超过 75 分钟的连续运动，需要外源碳水补充。

| 运动时长 | 补充建议 |
|---------|---------|
| < 60 分钟 | 无需补充，清水即可 |
| 60-90 分钟 | 视强度决定，低强度可不补 |
| > 90 分钟 | **30-60g 碳水/小时**（约每 30-45 分钟 1 条能量胶） |
| > 2.5 小时 | 可提升至 60-90g/小时（需多种碳水组合） |

**能量胶使用要点**：
- 提前 5 分钟服用（在感到疲劳前补充，而非感到疲劳时）
- 配合 150-200ml 水服用（勿干吃）
- 比赛：每 45 分钟一条（以 30g 碳水/条的标准胶为例）
- 训练时提前练习，找到个人最佳品牌和耐受性

## 个性化出汗率测量

**为什么重要**：出汗率因人而异（300-2000ml/小时），固定补水量不适合所有人。

**测量公式（ACSM）**：
```
每小时出汗量(ml) = [跑前体重(g) - 跑后体重(g) + 训练中饮水量(ml)] / 训练时间(小时)
```

**测量方法**：
1. 穿跑步装备称体重（克）
2. 完成 60 分钟中等强度跑，记录饮水量
3. 跑后立即称体重（穿同样装备，不排尿）
4. 代入公式计算

**AI 使用规则**：
- 有实测出汗率 → 用于制定个性化补水计划
- 无实测数据 → 使用通用建议（400-800ml/小时），标注「建议在下次 >20km 训练中测量」

---

## 电解质（钠）补充

**为什么重要**：长时间大量出汗后只补水不补钠 → 低钠血症（症状：头痛、恶心、严重时意识障碍）。

**建议**：
- 运动 >90 分钟：补充含钠饮料或盐丸（300-600mg 钠/小时）
- 常见参考：普通运动饮料约含 100-200mg 钠/250ml
- 重度出汗跑者（出汗后皮肤有明显白色盐渍）：钠需求更高

## 比赛日补给计划模板

| 时间节点 | 补给内容 |
|---------|---------|
| 赛前 2-3 小时 | 正常早餐（熟悉食物，勿尝试新东西） |
| 起跑前 30 分钟 | 150-200ml 水或运动饮料 |
| 跑步中（每 3-5km 设置站） | 水 150ml + 每 45 分钟 1 条能量胶 |
| 半程点附近 | 额外补一次盐丸（如有重度出汗） |
| 终点后 30 分钟内 | 碳水+蛋白质恢复餐 |

## 常见误区

| 误区 | 正确做法 |
|------|---------|
| 「口渴了再喝」 | 口渴时已轻度脱水，应提前定时补充 |
| 「只喝水，不喝运动饮料」 | >90 分钟需要电解质，纯水有低钠风险 |
| 「能量胶感觉太甜，不吃」 | 可换天然食物替代（香蕉、椰枣、米饭球） |
| 「比赛时第一次吃能量胶」 | 必须在训练中反复测试，找到耐受的品牌 |
